# Bakaan
Sistema de ventas para una conexión directa con clientes y consumidores finales con precios justos y sin intemediarios. Aunque se recomienda netbeans para editar estre proyecto, también se puede modificar desde un codespace (por ello el .devcontainer) pero muchas cosas están limitadas.

Se usa jframe, herencia y polimorfismo de clases, como parte del parcial de Programación orientada a objetos.

Creado por Juan Diego Guarnizo, Juan David Méndez, Tatiana Suaréz y Nicolás Rodríguez con mucho amor y cariño.
